# movieJava
### Function  
• 메인 페이지: 구현 페이지 전부에 접근 가능, 이미지 넘김 효과  
• 영화 검색: DB에 저장된 영화 검색  
• 회원 서비스: 가입, 로그인/아웃, 비밀번호/아이디 찾기, 마이페이지  
• 게시판: 게시글/댓글 작성, 조회, 수정, 삭제  
• 스토어: 상품 조회, 결제, 검색
